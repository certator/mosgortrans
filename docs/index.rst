================================
Mosgortrans parser documentation
================================

Mosgortrans parser is able to work with Mosgortrans.org_ and
Mosgortrans.ru_ schedule websites.

.. _Mosgortrans.org: http://mosgortrans.org/pass3/
.. _Mosgortrans.ru: http://www.mosgortrans.ru/rasp/

Required modules: Python 3.x, BeautifulSoup4_, `Enum module from Python 3.4`_.

.. _BeautifulSoup4: http://www.crummy.com/software/BeautifulSoup/
.. _`Enum module from Python 3.4`: https://pypi.python.org/pypi/enum34

Mosgortrans parser is hosted on GitHub_, and available under the terms of
BSD license.

.. _GitHub: https://github.com/mitya57/mosgortrans
